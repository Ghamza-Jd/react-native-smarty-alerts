# react-native-smarty-alerts

smarty alerts

## Installation

```sh
npm install react-native-smarty-alerts
```

## Usage

```js
import SmartyAlerts from "react-native-smarty-alerts";

// ...

SmartyAlerts.alert('title', 'message', 'ok', 'cancel', okHandler, cancelHandler);
```
